Lizard Database

Welcome to the Lizard Database created by Zach Attas as a project in Udacity's FullStack Web Developer Nanodegree.

Features:

  • Users can add their pet lizards to the database and their lizard's hobbies
  • Users can freely edit and delete their own entries, which are protected via Google or Facebook login
  • All entries are viewable; however, a user can only add entries or edit their own existing entries when logged in

Store your lizards and their hobbies to your hearts content!
